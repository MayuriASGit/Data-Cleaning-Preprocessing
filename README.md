 Data-Cleaning-Preprocessing
Titanic Data Preprocessing

This project focuses on cleaning and preparing the Titanic dataset for machine learning. The goal is to transform raw data into a structured format that can be used for building predictive models.
DataSet Link https://www.kaggle.com/datasets/yasserh/titanic-dataset

# What’s Done

1.Import the dataset and explore basic info (nulls, data types).
2.Handle missing values using mean/median/imputation.
3.Convert categorical features into numerical using encoding.
4.Normalize/standardize the numerical features.
5.Visualize outliers using boxplots and remove them.

# Tech Used

* Python
* Pandas
* NumPy
* Matplotlib & Seaborn

# Outcome

A clean and processed dataset ready for machine learning tasks like classification or prediction.

# How to Run
